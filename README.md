# Flappy Bird Python Game

## 📌 Project Overview

This project is a Python implementation of the classic **Flappy Bird** game. It was inspired by the open-source project from Clear Code Projects and serves as a learning exercise for game development using Python and Pygame.

Original reference code:
https://github.com/clear-code-projects/FlappyBird_Python/blob/master/flappy.py

---

## 🎮 Features

* Simple Flappy Bird gameplay mechanics
* Gravity and jump physics simulation
* Moving pipe obstacles
* Collision detection system
* Score tracking system
* Game restart after collision

---

## 🛠️ Technologies Used

* Python 3
* Pygame library

---

## 📚 What I Learned

* Game loop structure in Pygame
* Handling keyboard input events
* Collision detection techniques
* Sprite movement and animation
* Basic physics simulation (gravity & jump mechanics)

---

## 🚀 How to Run

1. Make sure Python is installed on your system
2. Install Pygame:

   ```bash
   pip install pygame
   ```
3. Run the game:

   ```bash
   python flappy.py
   ```
